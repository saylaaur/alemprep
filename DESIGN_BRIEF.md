## Продукт
AlemPrep — бесплатная веб-платформа подготовки к ЕНТ (гос-экзамен, Казахстан). Предметы:
математика, информатика, физика. Аудитория — школьники 16–18 лет, много из сёл, двуязычные
(казахский + русский). Настроение: спокойный фокус, «deep work», ничего лишнего.

## Референс и вайб
Как **oneprep.xyz**: минимализм, воздух, тёмная тема по умолчанию, аккуратная типографика,
никакой рекламы и визуального шума. Ощущение современного, быстрого, «умного» инструмента —
не школьного портала 2010-х. Клавиатурные шорткаты как first-class. Формулы через KaTeX.

## Стек (важно для реализуемости)
Next.js 15 + React 19 + **Tailwind CSS 3.4** + shadcn-стиль компонентов + CSS-переменные для
темы (light/dark) + шрифт Inter (кириллица). Иконки — lucide-react. Всё, что предлагаешь,
должно ложиться на Tailwind-утилиты и CSS-переменные — без экзотики, которую не собрать.

## Экраны (нужны макеты)
1. **Лендинг** — герой (что это + CTA «Начать бесплатно»), блок фич, превью предметов, футер.
2. **Дашборд** — приветствие, круговой прогресс дневной цели, стрик (огонёк), карточки
   предметов с % прогресса, блок «слабые темы».
3. **Тренажёр** — одна задача в фокусе: условие (с формулами), варианты 1–6, подсказка по
   шорткатам, прогресс-бар, кнопки «Проверить/Дальше», разбор после ответа. Есть focus-режим.
4. **Прогресс** — heatmap активности (как GitHub), сильные/слабые темы (полосы %), история сессий.
5. **Полный пробник** — экзамен под таймером: таймер всегда виден, навигатор вопросов
   (сетка номеров с состояниями), пометка «вернуться позже», экран результата с разбором.
6. **Мобильная версия** — верхний бар + off-canvas drawer навигации, тач-таргеты ≥44px.

## Что хочу от тебя
- Свежая, современная визуальная система: палитра (dark-first + light), типографическая шкала,
  отступы, радиусы, тени, состояния (hover/active/focus/disabled), акцентные цвета для
  success/warning (правильно/неправильно).
- Макеты 6 экранов выше (можно как HTML/Tailwind или чёткое описание + ключевые компоненты).
- Микро-детали, которые делают «дорого»: тонкие границы, мягкое свечение акцента, плавные
  переходы, состояние загрузки/пустоты.
- Двуязычие: макеты должны нормально выглядеть и на казахском (строки длиннее русских).

## Ограничения
- Только Tailwind-утилиты + CSS-переменные (мы потом маппим в `globals.css`/`tailwind.config.ts`).
- Без тяжёлых картинок/иллюстраций, которые сложно поддерживать. Иконки — lucide.
- Доступность: контраст AA, фокус-стили видимые (важно для клавиатурной навигации).

Верни: палитру и токены + макеты/описания экранов. Я перенесу в код точечно.
