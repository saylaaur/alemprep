## 1. TL;DR (30 секунд)

- **Что:** бесплатная веб-платформа для подготовки к ЕНТ (Казахстан), предметы: математика, физика, информатика, химия, биология, география.
- **Стек:** Next.js 15 + Supabase + Tailwind + next-intl (ru/kk).
- **Где:** на localhost, репо в GitHub (`saylaaur/alemprep`), не задеплоено.
- **Контент:** 33 опубликованных задачи по математике (9 ручных + 24 AI-сгенерированных). Остальные 5 предметов пустые.
- **Что сделано:** дизайн, тренажёр, локализация, AI-пайплайн контента, страница прогресса.
- **Что осталось:** мобильная адаптация, полный пробник, поддержка картинок в задачах, контент по 5 предметам, деплой.
- **До MVP-запуска:** ~3 недели работы + ~$10.
- **До полного запуска:** ~2–3 месяца + ~$30–50.

## 2. О продукте

Платформа в духе oneprep.xyz, но под казахстанский ЕНТ. **Принципы:**
- Минимализм, фокус, deep work
- Клавиатурные шорткаты (1–6 — выбор, Enter — проверить)
- Двуязычность ru/kk (казахский как первоклассный язык, не калька)
- Бесплатно навсегда для учеников
- Подсветка слабых тем, стрики, прогресс

**Целевая аудитория:** ~190 000 школьников/год сдают ЕНТ. Особый фокус — село + казахскоязычные (самый недообслуженный сегмент).

## 4. Стек и архитектура

### Технологии
- **Next.js 15** (App Router), React 19, TypeScript strict
- **Tailwind 3.4** + кастомная дизайн-система (CSS-переменные)
- **Supabase** (Postgres, Auth, RLS, SSR через `@supabase/ssr`)
- **next-intl 3.26**, `localePrefix: 'always'` → `/ru/...`, `/kk/...`
- **KaTeX** через CDN (auto-render для `$...$` и `$$...$$`)
- **Lucide-react** иконки
- **Google Gemini 2.5 Flash** для AI-пайплайна контента (free tier)

### Структура папок
```
app/[locale]/
  (app)/           — защищённые страницы (dashboard, subjects, practice, progress, full-practice)
  page.tsx         — лендинг
  login/, layout.tsx
components/        — ui/, layout/, practice/, math/, theme-*
lib/supabase/      — client/server/middleware, queries, auth-actions, practice-actions
types/db.ts        — типы по схеме БД (ручная поддержка)
i18n/              — routing.ts, request.ts
messages/          — ru.json, kk.json (паритет ключей обязателен)
supabase/migrations/ — 0001 schema, 0002 seed subjects/topics, 0003 seed logarithms
scripts/           — AI-пайплайн (transcribe + generate + variants)
middleware.ts      — next-intl + auth-guard
```

### БД (7 таблиц)
`profiles`, `subjects`, `topics`, `contexts`, `questions`, `sessions`, `attempts` — все под RLS.
`questions.body` — JSONB, полиморфно по типу: `single` / `multi` / `matching`.

## 10. Ключевые решения (важный контекст)

1. **AI-пайплайн на Gemini, не Anthropic.** Изначально код был на Anthropic SDK, переехали на `@google/genai` потому что у пользователя нет денег на Anthropic API. Free tier Gemini Flash покрывает наш масштаб. Если будет $25–50 — можно перейти на Claude Haiku ради качества.

2. **Сплит работы.** Cowork (этот ассистент) делает планирование/анализ/документы. Claude Code (CLI) делает кодинг. Передача через `TASKS.md` + ROADMAP.md + DESIGN.md.

3. **types/db.ts вручную.** Раньше был заглушкой, восстановлен по схеме из миграций. При изменении БД — обновлять руками или сгенерировать через supabase CLI.

4. **dev-сервер не проверяет типы.** SWC стрипает TS без проверки. Перед «готово» гонять `/verify` или `npm run typecheck`.

5. **Гонка с rate-limit Gemini.** Per-minute лимит ~15 запросов. На больших партиях ~30–50% запросов теряется как 503. Решается ожиданием 5–10 минут + повтором.

6. **Картинки пропускаются.** Без T11 четыре предмета (физика/химия/био/гео) теряют 40–60% задач. T11 — критический unlock.

7. **Запуск через школы и Telegram.** Маркетинг сначала органический, через свою школу (Dostyq School, Актобе) и подготовительные каналы.
