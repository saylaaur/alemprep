## 1. Где мы сейчас (июнь 2026)

### ✅ Готово

**Инфраструктура:**
- Next.js 15 + React 19 + TypeScript strict, Tailwind, shadcn-стиль
- Supabase (Postgres, Auth, RLS, SSR)
- Google OAuth, middleware-guard, защищённые маршруты
- next-intl: ru/kk, `localePrefix: 'always'`, паритет ключей
- KaTeX (через CDN, авто-рендер для `$…$` и `$$…$$`)
- Все ворота качества зелёные: `typecheck`/`lint`/`build`

**Продукт:**
- Лендинг (минимализм, hero, фичи, превью предметов, CTA, футер)
- Логин (Google OAuth, чистый дизайн)
- Дашборд (круговой прогресс цели, стрик, карточки предметов)
- Тренажёр (все 4 типа задач, шорткаты, попытки, стрики, focus-режим)
- Страница «Прогресс» (heatmap, слабые/сильные темы, история)
- Полная локализация (ru/kk) лендинга, дашборда, тренажёра

**Контент:**
- 12 тем по математике (seed-миграция)
- 33 задачи опубликовано (9 ручных + 24 сгенерировано через AI)
- AI-пайплайн рабочий: PNG → reference JSON → варианты → SQL (Gemini Flash, free tier)
- MathText умеет рендерить `type: 'latex'` блоки

**Качество:**
- Type-safety (после восстановления `types/db.ts`)
- Локализация callback + типобезопасные маршруты + ESLint flat-config
- Дизайн-система (цвета, тени, анимации, токены)

### ⏳ Не сделано

- Не задеплоено (только localhost)
- Нет мобильной адаптации
- Нет режима полного пробника
- Нет поддержки картинок в задачах (графики, схемы, карты)
- Информатика, физика, химия, биология, география — пустые
- Нет error tracking / analytics / тестов

## 3. Контент — план до банка задач

### Цели на 3 месяца подготовки ученика

| Предмет | Минимум | Достаточно | Хорошо |
|---|---|---|---|
| Математика | 250 | 500 | **800** |
| Информатика | 150 | 300 | **500** |
| Физика | 150 | 300 | **500** |
| Химия | 150 | 300 | **500** |
| Биология | 150 | 300 | **500** |
| География | 150 | 300 | **500** |
| **Всего** | **1050** | **2000** | **3300** |

Сейчас: **33 задачи** (1% от цели «хорошо»).

### Сколько материала собрать

Из одного пробника (~40 задач) при N=3 вариантов получается ~120 задач в БД.
Чтобы дойти до 3300:

| Цель | Эталонов | Пробников |
|---|---|---|
| 1050 (минимум) | ~350 | ~9 |
| 2000 (достаточно) | ~670 | ~17 |
| **3300 (хорошо)** | **~1100** | **~28** |

**Где брать пробники:**
- Официальные с testcenter.kz (раз в год)
- Прошлогодние ЕНТ (открыты)
- Сборники «Атамура», «Мектеп», «Шың кітап» (бумажные → скан/скриншот)
- Открытые архивы школьных репетиторов в Telegram
- Курсы Khan Academy → переводить условия (для базовой математики)

### Долевое распределение задач с картинками

| Предмет | Текстовые проходят | Нужен T11 |
|---|---|---|
| Математика | ~90% | редко |
| Информатика | ~90% | редко |
| Физика | ~60% | **критично** |
| Химия | ~55% | **критично** |
| Биология | ~50% | **критично** |
| География | ~45% | **критично** |

**Вывод:** математика и информатика можно гнать сейчас. Остальные 4 — после T11.

## 6. Рекомендуемая последовательность (по неделям)

### Неделя 1: Дожать ядро MVP

- T5 мобильная адаптация
- T7 полный пробник
- Деплой на Vercel + домен
- Анонс беты 5–10 друзьям, собрать первые сессии

**Цель:** платформа работает по публичному URL, друзья могут зайти с телефона.

### Неделя 2–3: Контент по математике

- Догнать математику до 500+ задач
- T11 поддержка картинок (готовим к остальным предметам)
- Презентация в одной школе

**Цель:** математика — полноценный банк. Видим первые сессии настоящих учеников.

### Неделя 4–6: Информатика и физика

- T12 сидинг тем информатики
- Генерация контента информатики до 300+
- Сидинг тем физики, генерация физики до 300+ (с картинками после T11)
- Telegram-посты, реферальные ссылки

**Цель:** активны 3 предмета, ~1500 задач в БД.

### Неделя 7–9: Химия, биология, география

- Миграции тем по 3 оставшимся предметам
- Контент каждый: ~150–300 задач
- T8 error tracking + T9 тесты — параллельно

**Цель:** активны все 6 предметов, ~2500 задач.

### Неделя 10–12: Полировка и продвижение

- Допиливание UX, баги, обратная связь
- Активный пиар: TikTok, Instagram Reels, школы
- Заявка на гранты
- Аналитика и оптимизация воронки

**Цель:** ~100 активных юзеров, первые отзывы, готовая платформа.

## 7. Определение готовности (Definition of Done)

Платформа готова к **публичному запуску**, когда:

- [x] ~~Деплоилась на публичном URL~~ → пока нет
- [ ] Мобильная версия работает (smoke-test на iOS Safari + Chrome Android)
- [ ] Google OAuth работает в проде
- [ ] Прохождение полного пробника работает end-to-end
- [ ] ≥500 задач по математике
- [ ] ≥250 задач по информатике
- [ ] ≥200 задач по физике, химии, биологии, географии (каждый)
- [ ] Страница «Прогресс» показывает реальную статистику
- [ ] Error tracking ловит ошибки
- [ ] Минимум 10 тестовых юзеров прошли по 1 сессии без блокеров

## 8. Самые большие риски

| Риск | Почему страшно | Что делать |
|---|---|---|
| Времени мало (учёба, ЕНТ-подготовка) | Можешь сам не успеть | Привлечь со-разработчика (Куатов?), фокус на 3 предмета вместо 6 |
| Контент в физике/химии без картинок неполный | Половина задач теряется | T11 обязателен, не пропускать |
| AI генерит дубликаты и кривой LaTeX | Качество страдает | Ручная проверка партиями, проще отбраковывать |
| Gemini rate-limit бьёт по производительности | Долго гонять | Либо терпеть, либо $25 на Claude Haiku |
| Никто не зарегистрируется после запуска | Демотивация | Школа + Telegram-каналы дают первых 50 человек гарантированно |

## 9. Главная вывод

**До MVP-запуска (математика + информатика, ~750 задач):** **~3 недели** при 1.5–2 ч/день, **~$10**.

**До полного запуска (все 6 предметов, ~2500 задач):** **~2–3 месяца** при том же темпе, **~$30–50**.

Бутылочное горлышко — **не код и не деньги**, а сбор материалов + ручная проверка контента.

Кодинг почти дошёл до финальной прямой. Дальше — операционный труд.
